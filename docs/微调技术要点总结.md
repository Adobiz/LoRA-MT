# 微调技术要点总结

> 比赛：AI 多语种翻译 | 模型：Hy-MT2-7B | 方式：LoRA + DoRA

---

## 六大技术要点

### 1. LoRA + DoRA 部分参数微调

**为什么**：全参数微调 7.5B 需 >100GB 显存（×3 optimizer states），不可行。

**实现**：
```python
peft_config = LoraConfig(
    task_type=TaskType.CAUSAL_LM,
    r=32,                    # 低秩维度
    lora_alpha=64,           # α = 2×r
    lora_dropout=0.05,
    target_modules=["q_proj", "v_proj", "k_proj", "o_proj"],
    use_dora=True,           # 权重分解 LoRA
    bias="none",
)
model = get_peft_model(model, peft_config)
```
- 仅训练 **27.6M / 7,532M 参数（0.37%）**
- adapter 仅 **110MB**

### 2. 五数据集混合训练

| 数据集 | 样本数 | 领域 |
|------|------:|------|
| translation_专业法律医疗 | 187K | 法律/外交 en/es/fr/ru→zh |
| translation_商务商务报告 | 141K | 商务报告 |
| translation_通用新闻类 | 125K | 新闻 10+语言→zh |

一次训练学到所有领域，通用数据占 14% 防灾难性遗忘。

### 3. 粤语特征字占位符保护

**问题**：粤语→普通话翻译需繁转简，但 OpenCC 会误杀粤语特征字。

**解决**：占位符保护法
```
原始: 杞人嘅朋友嘆咗一口氣
Step 1: 特征字→占位符  "嘅→__C06__", "咗→__C08__"
Step 2: OpenCC 繁→简  "嘆→叹"
Step 3: 占位符→恢复    "__C06__→嘅", "__C08__→咗"
结果: 杞人嘅朋友叹咗一口气  ← 粤语特征字保留
```

保护白名单（20字）：`係 邊 冇 裏 裡 嗰 哋 嘅 咗 啲 喺 佢 乜 嘢 冧 瞓 攰 氹 脷 嚟 冚`

### 4. 动态指令生成

根据 138 种语言的实际方向自动生成 prompt：
```python
if tgt == "zh":     instruction = f"将以下{src_name}文本翻译成中文："
elif src == "zh":   instruction = f"将以下中文文本翻译成{tgt_name}："
else:               instruction = f"将以下{src_name}文本翻译成{tgt_name}："
```

### 5. 训练稳定性保障

| 机制 | 配置 | 作用 |
|------|------|------|
| bf16 | True | 半精度，省 50% 显存 |
| gradient_checkpointing | True | 用计算换显存 |
| max_grad_norm | 1.0 | 梯度裁剪防爆炸 |
| warmup_ratio | 0.1 | 前 10% 步线性升温 |
| cosine scheduler | — | 余弦衰减，精细调整 |
